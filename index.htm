tmux

What is tmux?

  • A terminal multiplexer (I know this doesn't actually explain anything)
  • "Multiplexing" means combining a series of streams into a single conduit
  • tmux allows you to launch multiple windows inside a single terminal instance
  • Each window in itself allows you to open several sub-windows or panes

What is tmux?

The goal:

Providing better window management whilst working on the commandline.

3. Pair programming

Two programmers working together at one workstation, frequently switching between roles:

  • The driver writes code
  • The navigator reviews each line as it is typed in, and gives feedback

tmux's server/client architecture allows for remote pairing via SSH.

3. Pair programming

Pros

  • Minimal lag for the guest developer
  • SSH
  • Cross-platform (obviously excluding Windows)

Cons

  • Whilst pairing remotely, you are limited to the shell
  • Small learning curve for tmux navigation

Customization

Tmux' look and feel is highly customizable through a .tmux.conf file in your homedir.

Essentially, you can build your own development environment.
